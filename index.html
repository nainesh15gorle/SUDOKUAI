What is a CSP?

Sudoku is a classic Constraint Satisfaction Problem. Every empty cell is a variable, the digits 1–9 are its domain, and the row/column/box rules are the constraints.

Backtracking Search

The AI picks an empty cell, tries every digit, checks all constraints with isSafe(), and recurses. If it hits a dead-end it backtracks — undoing the last choice and trying the next.

MRV Heuristic

Minimum Remaining Values — the AI always picks the cell with the fewest legal digits first. This dramatically prunes the search tree and speeds solving.

isSafe() Constraints

  • No duplicate in any row
  • No duplicate in any column
  • No duplicate in any 3×3 box
